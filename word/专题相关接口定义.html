专题相关接口定义

1.专题-列表

接口名称：getSubjectList

专题按时间顺序降序排列，下滑可显示更多的

postData:

    {
        PageIndex: //---当前页码
        PageSize: //----每页显示多少条记录
    }

getData:

    {   
        List: [
            {
                SubjectId: //--专题ID
                Name: //-------专题标题
                SubName: //----标题简写
                ImageUrl: //---专题封面
            }
        ]
    }

2.专题-内容

接口名称：getSubjectInfo

这个接口是获取某个专题的一些基础信息

postData:

    {
        SubjectId: //--专题ID
    }

getData:

    {   
        SubjectId: //---------专题ID
        Name: //--------------专题标题
        SubName: //-----------标题简写
        ImageUrl: //----------专题封面
        Description: //-------专题描述

        ClubId: //------------圈子ID（这个圈子其实是后台在添加专题是推荐的圈子，每个专题只能推荐一个）
        ClubName: //----------圈子名称

        /*
        * 相关主题
        * 此处由编辑后台推荐相关的标签类别，可以是一级栏目页，二级栏目页或三级栏目页
        */
        CategoryList: [
            {
                CateId: //----栏目ID
                CateName: //--栏目名称
                ImageUrl: //--栏目封面图
            }
        ]
    }

接口名称：getSubjectArticle

这个接口是获取某个专题的帖子列表，可以分页

postData:

    {
        SubjectId: //--专题ID

        PageIndex: //--当前页码
        PageSize: //---每页显示多少条记录
    }

getData:

    {   
        ArticleList: [
            {
                ImageUrl: //----------图片地址
                Content: //-----------描述内容
                CollectNum: //--------被收藏的数量
                UserId: //------------用户ID
                UserImage: //---------用户头像 
                UserName: //----------用户名称
                Category:[ //---------帖子所属栏目名称（一级、二级、三级名称）不会跨栏目
                    {
                        CateId: //----栏目ID
                        CateName: //--栏目名称
                    }
                ]
            }
        ]
    }
