其他相关接口定义

1.聊天接口定义

默认显示当前1天内的聊天记录，每拉一次提前一天

接口名称：getListChat

postData:

    {   
        
        /*
        * 如果主人ID为空，该列表直接返回空数据
        */
        
        UserId: //-------对方的用户ID

        Header: {
            UserId: //--当前登录用户ID（未登录传空）
            Auth: //----当前登录用户Auth（未登录传空）
        }
    }

getData:

    {   
        ChatList: [
            {
                UpdateTime: //--聊天发表时间
                Content: //-----聊天内容
            }
        ]
    }
